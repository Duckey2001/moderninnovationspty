# Modern Innovations Pty Ltd

A modern, responsive company website built with React, Vite, and TailwindCSS.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Duckey2001/moderninnovationspty.git
cd moderninnovationspty
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 📦 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: TailwindCSS 4
- **Routing**: React Router DOM
- **Icons**: Lucide React
- **Language**: TypeScript

## 🌐 Deployment

This project can be deployed to various platforms:

### GitHub Pages
1. Build the project: `npm run build`
2. Deploy the `dist` folder to GitHub Pages

### Vercel/Netlify
- Connect your GitHub repository
- The platform will automatically detect and deploy the Vite project
